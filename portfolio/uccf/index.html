I work one day a week pro bono for UCCF: The Christian Unions, the charity which supports Christian Unions in universities across Great Britain.

I'm helping them to build a new, discrete authentication service, based on the OAuth specification, to allow UCCF's thousands of users to use their accounts outside of the main website.

This will make it possible for the organisation to move from a single monolith PHP application to discrete services and sites for different needs.
